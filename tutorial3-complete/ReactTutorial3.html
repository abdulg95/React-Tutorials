Learning React - Tutorial 03(Lists and Conditionals)

 

In react conditional logic is generally used to render dynamic content when certain conditions have been satisfied. For example if you have a POS app you might want to render tax or further useful information pertaining to the purchase when the user has entered the necessary details.

For a hands-on example here is an exercise from the react js homepage https://codepen.io/gaearon/pen/ZpVxNq?editors=0011. Try setting   <Greeting isLoggedIn={false} />  to true. What can you observe? Text is rendered on screen according to a condition that had to be satisfied.

Now that you are more familiar with conditionals in react lets put what we just learnt to use.
