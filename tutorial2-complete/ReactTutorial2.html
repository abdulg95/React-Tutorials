Learning React - Tutorial 02(State and Props)

 

In react, state is used to make components more dynamic. State and props are two ways react controls data flow of the app. The main difference between the two is that state is internal and should be controlled by the component itself while props are external and are passed in by what is rendering the component.

 Now here’s an example of how state is used in react. https://codepen.io/abdulg95/pen/NEqzMo.

State is an in built object in react which maps to a key of values in the example above our state is used to store a single key called count which is used to keep track of button clicks. To access the now defined state we use this.state .

To make changes to the state, call this.setState passing in an object map of keys to updated values. It is important to notice that when ever this.setState is called react re-renders the page , this is why multiple stateful components are usually frowned upon as this often leads to chaos.
